# Dice Roller

**Level:** Beginner

## 🎯 Goal
Simulate rolling one or more 6-sided dice.

## 🛠 Key Concepts & Skills
- Random Module
- Functions
- Looping

## 📋 Steps & Roadmap
1. Ask user for number of dice
2. Loop that many times
3. Generate random 1-6
4. Print results
