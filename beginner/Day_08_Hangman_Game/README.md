# Hangman Game

**Level:** Beginner

## 🎯 Goal
Build a graphical Hangman word-guessing game where players guess letters to uncover a secret word before running out of attempts, complete with a visual representation of the gallows.

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Drawing shapes on a Canvas widget (Tkinter Canvas)
- Random word selection from list/file sources
- Game state management (tracking guessed letters, lives)
- GUI dynamic keyboard controls

## 📋 Steps & Roadmap
1. Set up the main GUI window using the Black-Blue-White theme.
2. Initialize a Canvas widget (Black background) for drawing the gallows in Blue.
3. Select a random secret word and display underscores for each letter in White text.
4. Create an on-screen alphabet keyboard grid (Blue buttons, White text).
5. Implement button click handlers to process guesses, update word display, draw hangman parts, and disable guessed buttons.
6. Provide win/loss notification banner and a reset button.
