# Currency Converter

**Level:** Beginner

## 🎯 Goal
Convert an amount from one currency to another using fixed rates or an API.

## 🛠 Key Concepts & Skills
- Dictionaries (for rates)
- Math operations
- Input formatting

## 📋 Steps & Roadmap
1. Define exchange rates dict
2. Get amount and currencies
3. Calculate conversion
4. Print result
