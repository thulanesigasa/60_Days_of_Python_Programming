# URL Shortener

**Level:** Beginner

## 🎯 Goal
Map long URLs to short keys using a library like `pyshorteners` or an API.

## 🛠 Key Concepts & Skills
- External Libraries
- API interaction

## 📋 Steps & Roadmap
1. Install pyshorteners
2. Get long URL
3. Generate short URL
4. Print result
