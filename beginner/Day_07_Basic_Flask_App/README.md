# Basic Flask App

**Level:** Beginner

## 🎯 Goal
Build a basic web application using Flask that serves HTML pages styled using a CSS stylesheet adhering to the 60-20-10 theme.

## 🎨 GUI/Web Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Setting up a Flask application and development server
- Route rendering with HTML templates
- Serving static assets (CSS) in Flask
- Responsive layout styling with vanilla CSS

## 📋 Steps & Roadmap
1. Create the Flask directory structure (including `templates/` and `static/css/` directories).
2. Write a Python script (`app.py`) to initialize Flask and define routes.
3. Create an `index.html` template.
4. Write a CSS stylesheet in `static/css/style.css` implementing:
   - Dark background (`#121212` / `black`)
   - Cobalt blue navigational elements/buttons (`#1E3A8A` / `blue`)
   - White text highlights and body text (`#FFFFFF` / `white`)
5. Run the local development server and view the styled landing page in your browser.
