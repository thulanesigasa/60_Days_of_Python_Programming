# Basic Chatbot

**Level:** Beginner

## 🎯 Goal
A program that responds to specific user keywords.

## 🛠 Key Concepts & Skills
- String matching
- Conditionals
- While loops

## 📋 Steps & Roadmap
1. Loop for input
2. Check if input contains 'hello', 'bye', etc.
3. Print predefined response
4. Break on exit
