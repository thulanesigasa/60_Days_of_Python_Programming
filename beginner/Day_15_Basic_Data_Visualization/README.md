# Basic Data Visualization

**Level:** Beginner

## 🎯 Goal
Plot a simple graph from data points.

## 🛠 Key Concepts & Skills
- Matplotlib
- Lists/Arrays

## 📋 Steps & Roadmap
1. Install matplotlib
2. Define x and y lists
3. Plot line/bar chart
4. Show plot
