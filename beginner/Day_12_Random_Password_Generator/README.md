# Random Password Generator

**Level:** Beginner

## 🎯 Goal
Generate a strong password with a mix of characters.

## 🛠 Key Concepts & Skills
- String module constants
- Random module
- List joining

## 📋 Steps & Roadmap
1. Import string & random
2. Define character pool (letters, digits, symbols)
3. Select random N chars
4. Join and print
