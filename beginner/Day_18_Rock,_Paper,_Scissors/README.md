# Rock, Paper, Scissors

**Level:** Beginner

## 🎯 Goal
Play the classic game against the computer.

## 🛠 Key Concepts & Skills
- Random choice
- Game Logic (Conditionals)

## 📋 Steps & Roadmap
1. Define choices list
2. Get user choice
3. Get computer choice
4. Compare and determine winner
