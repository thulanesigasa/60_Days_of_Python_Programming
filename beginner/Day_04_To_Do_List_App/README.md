# To Do List App

**Level:** Beginner

## 🎯 Goal
Build a graphical To-Do List application where users can add, view, and delete tasks.

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Tkinter Listbox and Scrollbar widgets
- Storing task items in memory (lists)
- File I/O for saving and loading tasks
- Handling selection events in Tkinter

## 📋 Steps & Roadmap
1. Set up the main GUI window styled with a Black background.
2. Add a Listbox to display tasks (Black/Dark Blue background, White text).
3. Create an Entry widget for typing new tasks.
4. Implement the "Add Task" button to insert tasks into the listbox.
5. Implement the "Delete Task" button to remove the selected task.
6. Save the task list to a text file automatically when changes are made.
