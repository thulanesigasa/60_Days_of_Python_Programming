# Flashcard Quiz App

**Level:** Beginner

## 🎯 Goal
Display a question (or word) and ask for the answer/definition.

## 🛠 Key Concepts & Skills
- Dictionaries (Question: Answer)
- Loops
- Scoring

## 📋 Steps & Roadmap
1. Create dictionary of terms
2. Loop through items
3. Print key, ask for value
4. Check answer and score

## 🔗 Resources & Hints
- Google is your friend! Try searching for the key concepts.
