# Guess the Word Game

**Level:** Beginner

## 🎯 Goal
Similar to Hangman but maybe with hints or definitions.

## 🛠 Key Concepts & Skills
- String manipulation
- File I/O (load words)

## 📋 Steps & Roadmap
1. Load words from file
2. Pick random word
3. Show hint
4. Get guess
5. Check correctness

## 🔗 Resources & Hints
- Google is your friend! Try searching for the key concepts.
