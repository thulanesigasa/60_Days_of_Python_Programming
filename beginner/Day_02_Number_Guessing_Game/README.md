# Number Guessing Game

**Level:** Beginner

## 🎯 Goal
Build a graphical user interface (GUI) application where the computer selects a random number, and the player tries to guess it. The application should provide feedback on whether the guess is too high, too low, or correct.

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Random number generation using the `random` module
- Tkinter layout managers
- Input handling and validation
- Dynamically modifying widget text (feedback label)

## 📋 Steps & Roadmap
1. Generate a random target number between 1 and 100 on startup.
2. Build the main GUI window with an input field for guesses, a submit button, and a feedback label.
3. Apply the Black-Blue-White theme to the interface.
4. Implement a callback function to compare the user's guess against the target number, counting attempts and updating the feedback label.
5. Provide a reset option to start a new game.
