# Text-Based Adventure Game

**Level:** Beginner

## 🎯 Goal
A story where the user makes choices that affect the outcome.

## 🛠 Key Concepts & Skills
- Nested Conditionals
- Functions for scenes
- Global state/variables

## 📋 Steps & Roadmap
1. Design story flow
2. Create functions for rooms/scenes
3. Get user choice
4. Branch to next function
