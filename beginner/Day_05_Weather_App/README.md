# Weather App

**Level:** Beginner

## 🎯 Goal
Build a graphical weather application that fetches real-time weather information for a user-specified city using an API (such as OpenWeatherMap).

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Sending HTTP requests using the `requests` library
- Parsing JSON data
- Integrating external APIs
- Handling network connection and API exceptions gracefully

## 📋 Steps & Roadmap
1. Sign up and obtain an API key from a free weather service (e.g., OpenWeatherMap).
2. Set up the main GUI window using the Black-Blue-White theme.
3. Add a Search bar (Entry widget) for entering the city name.
4. Implement a "Get Weather" button that makes an asynchronous/synchronous API request.
5. Parse the JSON response to extract temperature, humidity, and description.
6. Display the weather results using stylized labels in the interface.
