# Alarm Clock

**Level:** Beginner

## 🎯 Goal
Run a timer and play a sound when time is up.

## 🛠 Key Concepts & Skills
- Time/Datetime module
- While loop
- Audio library (playsound)

## 📋 Steps & Roadmap
1. Get target time
2. Loop and check current time
3. When match, print Wake Up and play sound
