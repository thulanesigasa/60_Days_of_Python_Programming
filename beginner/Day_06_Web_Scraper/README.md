# Web Scraper

**Level:** Beginner

## 🎯 Goal
Build a python tool (with a simple GUI interface to trigger operations and display logs) that scrapes data from a website and saves it to a CSV file.

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Web scraping basics using `BeautifulSoup` and `requests`
- Fetching and parsing HTML structures
- Saving structured data into CSV files using the `csv` module
- Creating a simple control dashboard UI to specify target URL and view logs

## 📋 Steps & Roadmap
1. Set up the control dashboard using the Black-Blue-White theme.
2. Provide a text field (Entry widget) for entering the target URL.
3. Implement a "Start Scraper" button to fetch the target HTML.
4. Parse the HTML using `BeautifulSoup` to locate relevant content tags (e.g., titles, prices, articles).
5. Output live status logs to a scrollable text area on the GUI.
6. Export the scraped data to a `.csv` file.
