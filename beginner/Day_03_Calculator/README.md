# Calculator

**Level:** Beginner

## 🎯 Goal
Build a graphical calculator application capable of performing basic mathematical operations (addition, subtraction, multiplication, division).

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Handling button grids using Tkinter Grid manager
- Building an expression evaluation engine
- State management and display updates
- Exception handling (e.g., division by zero)

## 📋 Steps & Roadmap
1. Design the layout using rows and columns for display and buttons.
2. Initialize the display area with the correct background colors.
3. Write functions to append numbers and operators to the current expression.
4. Implement the evaluation function triggered by the `=` button.
5. Add clear (`C`) and backspace functionality.
6. Set foreground/background colors of buttons to reflect the 60-20-10 theme.
