# Hello World GUI

**Level:** Beginner

## 🎯 Goal
Build a simple graphical user interface (GUI) application that displays "Hello, World!".

## 🎨 GUI Theme (60-20-10 Rule)
This project follows a strict color harmony guideline to ensure a clean and professional appearance:
- **60% Dominant (Background):** Black / Dark Theme (`#121212` or `#000000`)
- **20% Secondary (Containers & Controls):** Deep/Cobalt Blue (`#1E3A8A` or `#2563EB`)
- **10% Accent (Highlights & Text):** Pure White/Ice Blue (`#FFFFFF` or `#E0F2FE`)

## 🛠 Key Concepts & Skills
- Introduction to GUI frameworks (e.g., Tkinter)
- Creating a main application window
- Label and Button widgets
- Window event loops

## 📋 Steps & Roadmap
1. Initialize the GUI window and set its size/title.
2. Add a label displaying "Hello, World!" and style it according to the GUI theme.
3. Start the main event loop to display the window.
