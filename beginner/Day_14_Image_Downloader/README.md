# Image Downloader

**Level:** Beginner

## 🎯 Goal
Download an image from a URL and save it to disk.

## 🛠 Key Concepts & Skills
- Requests library
- File I/O (binary write)

## 📋 Steps & Roadmap
1. Get URL from user
2. Request content
3. Open file in 'wb' mode
4. Write content to file
