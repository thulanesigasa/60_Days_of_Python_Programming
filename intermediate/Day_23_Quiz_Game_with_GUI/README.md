# Quiz Game with GUI

**Level:** Intermediate

## 🎯 Goal
A multiple-choice quiz using Tkinter or PyQt.

## 🛠 Key Concepts & Skills
- GUI Widgets (Radiobuttons, Buttons)
- Classes for UI

## 📋 Steps & Roadmap
1. Design Question class
2. Setup GUI window
3. Load question
4. Check answer on button click
5. Show score
