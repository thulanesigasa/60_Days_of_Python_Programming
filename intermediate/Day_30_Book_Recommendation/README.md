# Book Recommendation Project

**Level:** Intermediate

## 🎯 Goal
Suggest books based on genre or similarity.

## 🛠 Key Concepts & Skills
- Pandas
- Basic Filtering/Similarity

## 📋 Steps & Roadmap
1. Get dataset (CSV)
2. Load into Pandas
3. Filter by user input (Genre/Author)
4. Return top rated matches
