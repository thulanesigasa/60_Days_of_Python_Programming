# Chat Application

**Level:** Intermediate

## 🎯 Goal
A simple client-server chat using Sockets.

## 🛠 Key Concepts & Skills
- Socket Programming
- Threading

## 📋 Steps & Roadmap
1. Create Server script (bind, listen)
2. Create Client script (connect, send)
3. Use threads to handle receiving messages continuously
