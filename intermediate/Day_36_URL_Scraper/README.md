# URL Scraper

**Level:** Intermediate

## 🎯 Goal
Extract all links from a webpage.

## 🛠 Key Concepts & Skills
- Regular Expressions or BeautifulSoup
- HTML <a> tags

## 📋 Steps & Roadmap
1. Fetch page
2. Find all href attributes
3. Filter valid URLs
4. Save to file
