# Simple Blog System

**Level:** Intermediate

## 🎯 Goal
A web app to create, read, update, and delete (CRUD) posts.

## 🛠 Key Concepts & Skills
- Flask/Django
- Databases (SQLAlchemy)
- REST principles

## 📋 Steps & Roadmap
1. Setup database model
2. Create routes for CRUD
3. Build HTML forms
4. Display posts list
