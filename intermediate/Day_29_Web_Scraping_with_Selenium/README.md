# Web Scraping with Selenium

**Level:** Intermediate

## 🎯 Goal
Scrape a dynamic website that requires JavaScript or Login.

## 🛠 Key Concepts & Skills
- Selenium WebDriver
- Browser Automation

## 📋 Steps & Roadmap
1. Install Selenium & Driver
2. Open browser
3. Interact with elements (click, type)
4. Extract data
