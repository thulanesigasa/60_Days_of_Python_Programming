# File Organizer

**Level:** Intermediate

## 🎯 Goal
Script to sort files in a folder into subfolders by extension.

## 🛠 Key Concepts & Skills
- OS module
- Shutil module
- Path manipulation

## 📋 Steps & Roadmap
1. Scan directory
2. Check file extension
3. Move to corresponding folder (Images, Docs, etc.)
