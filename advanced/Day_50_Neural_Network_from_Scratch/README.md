# Neural Network from Scratch

**Level:** Advanced

## 🎯 Goal
Build a NN without frameworks to understand the math.

## 🛠 Key Concepts & Skills
- Matrix Math (NumPy)
- Backpropagation
- Gradient Descent

## 📋 Steps & Roadmap
1. Define layers
2. Forward pass
3. Calculate loss
4. Backward pass (update weights)
