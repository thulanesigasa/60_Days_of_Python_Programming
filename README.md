# 60 Days of Python Programming

This repository contains a comprehensive 60-day journey through Python programming, spanning from beginner to advanced levels. Each day focuses on a specific project or concept to build professional Python skills.

## Project Structure

The projects are organized into three main levels:

### 🟢 Beginner (Days 1–20)
Focuses on core Python concepts, basic GUIs, and fundamental automation tools.
- **Day 01**: Hello World GUI
- **Day 05**: Weather App
- **Day 07**: Basic Flask App
- **Day 13**: Basic Chatbot
- **Day 15**: Basic Data Visualization
- **Day 17**: URL Shortener
- ... and more.

### 🟡 Intermediate (Days 21–40)
Introduces more complex applications, APIs, web scraping, and data management.
- **Day 21**: Personal Portfolio Website
- **Day 24**: Chat Application
- **Day 28**: Tic Tac Toe Game with AI
- **Day 31**: RESTful API with Flask
- **Day 33**: Interactive Map
- **Day 40**: Machine Learning Model Deployment
- ... and more.

### 🔴 Advanced (Days 41–60)
Covers cutting-edge topics like AI, Blockchain, Computer Vision, and full-stack development.
- **Day 41**: Social Media Dashboard
- **Day 42**: E-commerce Website
- **Day 45**: Blockchain Implementation
- **Day 50**: Neural Network from Scratch
- **Day 56**: Real-Time Object Detection
- **Day 60**: Self-Driving Car Simulation
- ... and more.

## How to Use This Repo

1. Clone the repository.
2. Navigate to the level you are interested in (`beginner`, `intermediate`, or `advanced`).
3. Explore the daily projects and run the corresponding Python scripts.
